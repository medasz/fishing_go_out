# 钓鱼拦截 - Chrome 浏览器威胁情报扩展

实时检测访问网站的威胁情报，当访问钓鱼/恶意域名时弹出告警。

## 功能特性

- **实时域名检测**：每次网页导航时自动查询威胁情报
- **双重情报源**：
  - **URLhaus** (abuse.ch) — 全球恶意软件分发 URL 数据库
  - **本地启发式检测** — 检测品牌仿冒、DGA 域名、可疑 TLD 等
- **页面内告警横幅**：检测到威胁时在当前页面顶部显示红色告警
- **桌面通知**：通过 Chrome 通知系统告警
- **扩展徽章**：工具栏图标显示当前页面安全状态
- **检测统计**：Popup 弹窗展示累计检测数据
- **缓存机制**：30 分钟缓存避免重复 API 请求

## 安装方法

1. 打开 Chrome 浏览器，地址栏输入 `chrome://extensions/`
2. 打开右上角 **开发者模式** 开关
3. 点击左上角 **加载已解压的扩展程序**
4. 选择 `fishing_go_out` 文件夹
5. 扩展安装完成，工具栏会出现盾牌图标

## 使用方式

- **自动检测**：安装后无需操作，每次访问网站自动检测
- **查看状态**：点击工具栏图标查看当前页面安全状态和统计
- **忽略告警**：点击告警横幅上的「我了解风险，继续访问」可关闭告警
- **返回安全**：点击「返回安全页面」回到上一页
- **手动检测**：Popup 窗口会自动查询当前页面对应的域名信息

## 威胁情报源

### URLhaus (abuse.ch)
- 由瑞士 abuse.ch 运营的非营利威胁情报平台
- 收集全球恶意软件分发 URL
- 免费使用，无需 API Key
- 查询端点: `https://urlhaus-api.abuse.ch/v1/host/`

### 本地启发式检测
- 检测知名品牌仿冒域名 (typosquatting)
- DGA 生成域名识别
- 可疑顶级域名检测 (`.tk`, `.ml`, `.xyz` 等)
- 域名异常特征分析

## 注意事项

- 首次访问某域名时会发起 API 请求，可能稍慢
- 安全域名会缓存 30 分钟，避免重复查询
- URLhaus API 有限速，请勿短时间内大量查询
- 建议搭配广告拦截器使用更佳
